# An Insight into Extreme Learning Machines: Random Neurons, Random Features and Kernels

## Why read now

- Good bridge between the original ELM papers and the broader theory story. Read this once the implementation basics are clear.

## Claim

- Interprets ELM more broadly through random neurons, random features, and kernel methods, and argues for a unifying view of these approaches.

## Method

- Theory-oriented discussion rather than a pure implementation paper; connects ELM to SVM and related feature-space methods.

## Evidence

- Emphasis is on conceptual and theoretical framing rather than only benchmark comparisons.

## Relevance to this repo

- Helpful for the theory chapter when explaining what kind of model family ELM belongs to.
- Useful for framing the difference between practical implementation choices and broader theoretical claims.
